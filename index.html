Was du tun musst

1. Diese zwei Dateien in dein GitHub-Repo legen: - index.html - app.js 2. GitHub Pages aktivieren 3. In Supabase RLS aktivieren für: - public.items - public.item_fixed_properties - public.item_choice_groups - public.item_choice_group_options 4. Policies für authenticated auf: - select - insert - update - delete setzen 5. Mit deinem Supabase-Auth-Benutzer einloggen Diese Version kann: - Items anzeigen - per Klick laden - Item ändern - Item löschen - item_fixed_properties bearbeiten - item_choice_groups bearbeiten - item_choice_group_options bearbeiten - beim Anlegen automatisch - items - item_fixed_properties - item_choice_groups - item_choice_group_options schreiben
